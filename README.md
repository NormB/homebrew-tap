# homebrew-tap

Homebrew tap for [sipnab](https://github.com/NormB/sipnab) — a SIP & RTP capture, analysis, and security tool.

## Install

```sh
brew install NormB/tap/sipnab
```

or

```sh
brew tap NormB/tap
brew install sipnab
```

Provides prebuilt binaries for macOS on both Intel and Apple Silicon.

For Linux, use the `.deb` packages (`amd64` / `arm64`) or the static `musl` binaries from the [sipnab releases](https://github.com/NormB/sipnab/releases).
